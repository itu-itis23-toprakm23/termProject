About Me Memory Game

Welcome to Memory Game!

You can start the game by simply clicking 'Start Game' button. After that, letters of my name -'M','U','R','A','T'- will be randomly shuffled and you have 2 seconds before they face down. When they face down, you need to click on letters in the correct order of my name. If you make a wrong attempt, you lose the game.
